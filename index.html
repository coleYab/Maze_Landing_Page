Maze

A simple 3D game with raycasting and C.

View Project Repository

Key Features

Raycasting

This project is made possible with raycasting
that will make the game efficent and fast to
render items

Custom Map

The user can design its own game world for this
game, that will make the user to design its
own complexity of the maze.

Custom game textures

The user can simply specify which texture to use
for each walls this will improve the feel of
the game.

About

This maze game was inspired by my curiosity about game
development and is part of my project for Holberton
School. I wanted to explore the intricacies of
creating an engaging and challenging game from
scratch. This project allowed me to apply my skills
in programming, design, and problem-solving. I hope
you enjoy playing it as much as I enjoyed building
it.
